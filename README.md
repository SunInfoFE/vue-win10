### 功能

- [x] 消息栏的展示隐藏
- [x] 消息栏的消息单条删除、全部清除

- [x] 任务栏的显示桌面功能
- [x] 任务栏的时间显示

- [x] 桌面快捷方式的显示
- [ ] 桌面快捷方式的拖拽排序
- [x] 桌面快捷方式双击打开窗口

- [x] 窗口的关闭
- [x] 窗口的最大化及恢复
- [x] 窗口的最小化
- [x] 窗口的resize
- [x] 窗口的remove
- [x] 窗口的层叠处理
- [x] 窗口和任务栏的互动

- [ ] 菜单 By@haise

### 技术点

* 使用了 webpack 实现了 代码分离、动态导入。
